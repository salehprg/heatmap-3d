# Heatmap 3D

install and run with Docker.
```
docker build -t heatmap-3d .
docker run -p 8282:8282 --name heatmap-3d -d heatmap-3d
```
verify the container is running.
```
docker ps
```

open at `http://<server-ip>:8282`
